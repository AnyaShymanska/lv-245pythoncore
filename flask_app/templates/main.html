My address book — вбудована прикладна програма для збереження контактної інформації, створена компанією Apple Inc. Входить до стандартної поставки Mac OS X. Представляє собою централізоване сховище контактної інформації. Характерною рисою даної програми є інтеграція та синхронізація з багатьма іншими прикладними програмами та функціями macOS. Після виходу OS X Mountain Lion програма отримала назву Contacts (Контакти). Остання версія комп'ютеризованої адресної книги, включеної в MacOS, - Contacts 10.0

Можливості програми:

  • Інтеграція з більшістю прикладних програм для Mac OS X, з Mail, iCal, iChat, Safari, FaceTime та іншими.
  • Синхронізація з КПК, iPod, iPhone, іншим Mac’ом або з Інтернет-порталами — MobileMe, Yahoo!, Google.
  • Надання стороннім розробникам API для доступу до адресної книги користувача.
  • Підтримка імпорту та експорту контактів у форматі vCard.
  • Наявність C та Objective-C програмного інтерфейсу.
  • Автоматичне форматування номерів телефонів за заданими шаблонами.
  • Відображення адрес контактів на Google Maps.
  • Систематизація контактів: папки, групи, смарт-групи.
  • Пошук контактів або контактної інформації.
  • Сортування контактів: за іменем або прізвищем.
  • Архівування адресної книги.
  • Картки адресної книги можуть відображати файли наступних форматів: JPEG, GIF, TIFF, PNG і PDF.
